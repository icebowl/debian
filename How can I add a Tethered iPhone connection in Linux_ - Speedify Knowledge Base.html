How can I add a Tethered iPhone connection in Linux?

Speedify can use USB tethered iPhones to add a cellular internet connection to your Linux device. To use this, your iPhone's cellular plan must support hotspots/tethering.

Prerequisites

Usually you need to install additional packages to support tethering iPhones to Linux. The exact packages can vary between distributions and versions.

For Ubuntu, the  ipheth-utils package is needed for tethering iPhones ("sudo apt install ipheth-utils")

For Raspbian, the  usbmuxd package adds support for tethering iPhones ("sudo apt install usbmuxd")

If you are using another distribution, try the packages mentioned above if they are available. Otherwise, search online for the specific package needed for your distribution.

Tethering

To start tethering:

  1. Plug the iPhone to the computer via USB
  2. On the iPhone go to Settings > Cellular > Set Up Personal Hotspot, and turn on USB Tethering.
  3. In Speedify, within a few seconds, you will see a new "bubble" appear representing the tethered phone.

Once it appears, you can adjust the Connection Settings, such as Priority, to control how Speedify uses the adapter.
